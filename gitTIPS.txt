common commands to upload using git

git init   //initiilize the git files

git status     // check the current status of your git  upload


git add .    // add all of the current uncommitted files to your local git save


git commit -m "commit notes"     //commit your current files that you have added to your local save 


git remote add origin https://github.com/coltonrohanhub/tempTest.git     // add the origin file for your remote hub


git push -u origin main     // final push onto githubs servers

git clone __repository__       // clone a copy of an online reposiroty onto your local computer

/// branching and merging 

git branch __name__      // make a branch of your current project that you can later merge if you want
git branch               // check what branches are currently associated with your project
